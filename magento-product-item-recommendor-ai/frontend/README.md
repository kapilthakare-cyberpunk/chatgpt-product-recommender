# Magento Product Recommender AI - Frontend

This is the frontend for the Magento Product Recommender AI application, built with Next.js.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:3000`.

## Features

- Input field to enter a Magento item ID
- Fetches recommendations from the backend API
- Displays recommended products in a responsive grid
- Loading indicator during API requests
